# tglab_test
Repository for TGLab test assignment project

1. UART + GPS parsing.
   Run start_uart_gps.sh.

2. Video Stream.
   Run start_stream.sh. 
   We assume that gstreamer, gstreamer rtsp server and rpicam tools are installed...

3. ROS-2 node for UART.
   Run start_ros2_node.sh.
   I used Dashing for this test assignment...
